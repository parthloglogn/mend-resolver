# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

## Security Practices

- All dependency versions are verified on Maven Central before application
- No dependency downgrades are ever performed
- All changes are compatible with JDK 17
- CVE fixes are sourced exclusively from user-provided Mend reports
- Git branches are created for all changes to enable review

## Reporting Security Issues

Please report security vulnerabilities by opening an issue in the repository.
